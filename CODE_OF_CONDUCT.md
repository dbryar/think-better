# Code Of Conduct

This project exists to make people harder to manipulate, not easier to recruit.

Do not use project spaces for harassment, hate, threats, spam, bad-faith flooding, or recruitment into extremist, conspiratorial, cultic, or violent ideologies. Criticism of the project is welcome when it is specific and useful.

Moderation decisions should preserve a public, inspectable, correctable project. They should not be used to protect contributors from legitimate correction.
